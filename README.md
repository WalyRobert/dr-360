# DR 360° - Dodge Recian

Um player de vídeo 360° de alta performance usando Three.js e React. Suporta upload de arquivos locais e URLs remotas com interface imersiva, head tracking, modo VR, loop, mute e mais.

## Features
- Renderização 3D imersiva com Three.js
- Controles de mouse/touch para navegação
- Head tracking via DeviceOrientation
- Modo VR (stereo para cardboard)
- Modos de visualização: 360°, 180°, 120°
- Auto-hide controls e auto-center
- Progress bar, zoom, playback speed

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:  
   `npm install`

2. Run the app:  
   `npm run dev`
